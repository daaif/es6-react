Cahrgement de données avec D3

D3 présente un ensemble de fonctions permettant d'acquérir les données à partir du réseau. Toutes ces fonctions utilisent l'API fetch qui se base sur le modèle des promesses.

Plusieurs parseurs sont disponibles pour récupérer et parser à la volée des données de types CSV, TSV, XML, JSON etc.

  • d3.xml()
  • d3.csv()
  • d3.tsv()
  • d3.json()
  • d3.image()
  • d3.text()
